What is Javascript

JavaScript is a high level programming language.

Properties of JavaScript

1. Loosely Typed 2. Can be used in both the Front End and Back End. 3. Used for DOM (Document Object Model) Manupulation.

JavaScript Linking Methods

There are three methods using which the Javascript can be added to a HTML webpage.

1. Inline Script : JavaScript can be directly added to the HTML file : 2. Internal Script : JavaScript code can be placed within the script tag within the HTML file itself, within the head or body section of the HTML file. 3. External Script : JavaScript code can be placed inside an external file with a .js extension and then can be linked to HTML file using script tag using src attribute.
