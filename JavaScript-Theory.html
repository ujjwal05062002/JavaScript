What is Javascript

JavaScript is a high level programming language.

Properties of JavaScript

1. Loosely Typed
2. Can be used in both the Front End and Back End.
3. Used for DOM (Document Object Model) Manupulation.

JavaScript Linking Methods

There are three methods using which the Javascript can be added to a HTML webpage.

1. Inline Script : JavaScript can be directly added to the HTML file :
2. Internal Script : JavaScript code can be placed within the script tag within the HTML file itself, within the head or body section of the HTML file.
3. External Script : JavaScript code can be placed inside an external file with a .js extension and then can be linked to HTML file using script tag using src attribute.

Identifier Defination :

Name of programming entities like Variable, Object, Classes, etc.

Rules of Identifiers :

1. We can't use any Keyword.
2. Digit can't be used in prefix.
3. We can't use any special character except underscore.
4. We can't use same name for more than one time under the same statement block.

DataTypes in JavaScript

1. Number: Integer and Float
2. Boolean : True and False
3. String : Text between " "
4. Null : 0
5. Undefined : Not Defined
6. object : obj = []
7. BigInt : Integer with High Value
8. Array : [X1, X2, X3, X4]

Data Representation

Variable

1. Used when data is fixed.
2. Variables are name of memory location in memory.
3. Variables are placeholder for data in RAM.
4. Syntax : 1. var -> Functional Level
2. let -> Block Level
3. const -> Block level

Constant Literals

constant literals are used when the data is fixed. 1. Var a = 10 (Integer Number)
2. Var b = true or false (boolean)
3. Var c = "string"
4. Var d = 20.23(Float Number)
